# Contributing to sui-gas-station

Thanks for your interest in contributing! This project is Apache-2.0 licensed and welcomes contributions.

## Getting Started

```bash
git clone https://github.com/Danny-Devs/sui-gas-station.git
cd sui-gas-station
pnpm install
pnpm test
```

## Submitting Changes

1. Fork the repo and create a branch from `main`
2. Make your changes
3. Add or update tests as needed
4. Ensure `pnpm test` and `pnpm typecheck` pass
5. Open a PR with a clear description of what changed and why

## Reporting Bugs

Open an issue with:
- What you expected to happen
- What actually happened
- Steps to reproduce
- Node version and OS

## Code of Conduct

Be respectful. We're all here to build something useful.
